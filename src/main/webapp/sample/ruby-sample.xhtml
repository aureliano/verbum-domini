Ruby

This example shows how you can consume service using only Ruby core libraries. You may be interested about using any other community free or third party libraries (gems).

Sample

require 'net/http'
require 'uri'

uri = URI.parse('http://server-host/verbumdomini/apirest/bibles')
http = Net::HTTP.new(uri.host, uri.port)

# Consume service as XML.
request = Net::HTTP::Get.new(uri.request_uri)
request.add_field "Accept", "application/xml"
response = http.request(request)

puts response.body

# Consume service as JSON.
request = Net::HTTP::Get.new(uri.request_uri)
request.add_field "Accept", "application/json"
response = http.request(request)

puts response.body
