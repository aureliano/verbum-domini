Python

This example shows how you can consume service using only Python core libraries. You may be interested about using any other community free or third party libraries (eggs).

Sample

import urllib2

url = 'http://server-host/verbumdomini/apirest/bibles'

""" Consume service as XML. """
request = urllib2.Request(url)
request.add_header('Accept', 'application/xml')
content = urllib2.urlopen(request).read()

print content

""" Consume service as JSON. """
request = urllib2.Request(url)
request.add_header('Accept', 'application/json')
content = urllib2.urlopen(request).read()

print content
