Verbum Domini

Chapter

The chapter resource provides services for getting available book chapters. You can navigate through other services from here. That means you can fetch all available chapters with the URL /chapters or a specific chapter by requesting /chapters/{chapterId}, or even navigating through nested services like /chapters/{chapterId}/verses/{verseId}.

Services

The services you can access from here are listed bellow.

Chapters list

Fetch the chapters list.

URL/chapters
MethodGET
Content-typeapplication/xml, application/json
Results200 OK, XML|JSON

Parameters

Name Description Type Default
start The first element to return (> 0) integer 1
pages The number of pages to return (> 0 <= 2) integer 1

Headers

Name Description Type Default
Accept The content-type that will be acceptable in response string application/xml
