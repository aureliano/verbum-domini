About the API

This system is an open data RESTFul API, which provides a set of services for retrieving data about bibles translations. Such API let you easily navigate through resources. The availability of those services promotes creation of applications for many platforms like Android, iOS, desktop applications or even web sites using your preferred set of development tools. And here you're gonna find some insights about this API.

Consuming services

In order to consume data, all you need is a web browser or even build an application that feet your needs. Data returned by the services are provided as XML or JSON. You can choose between then by just setting header property "Accept". An example using cURL to consume service which return data as XML would be:

curl --header "Accept:application/xml" http://path/to/service


For those using Java to create a client application to consume data it may be interesting to check that Java interface for searching through services.
