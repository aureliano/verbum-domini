The annotation resource provides services for getting available chapter annotations.

Annotations list

Fetch the annotations list.

URL/annotations
MethodGET
Content-typeapplication/xml, application/json
Results200 OK, XML|JSON

Parameters

Name Description Type Default
start The first element to return (> 0) integer 1
pages The number of pages to return (> 0 <= 2) integer 1

Headers

Name Description Type Default
Accept The content-type that will be acceptable in response string application/xml
